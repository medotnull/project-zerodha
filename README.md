This is a simple backend project that simulates Zerodha’s order book functionality. It lets you place buy/sell orders, track pending/completed orders, and view real-time order status - perfect for learning trading system basics with Node.js.

Tech Stack
•	Backend: Node.js + Express.js
•	Database: PostgreSQL with Prisma ORM
•	Real-time: WebSocket (or Socket.io)
•	Auth: JWT + bcrypt
•	Testing: Jest
